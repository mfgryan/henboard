# Scrum
A simple scrum board using node.js

# Getting started

**1st method**
Clone the repo and run **node index.js** 
optional arguments hostname and port: **node index.js 0.0.0.0 8000"**

**2nd method**
Clone the repo and run docker-compose up --build -d

# Requirements
node.js
Docker (optional)
