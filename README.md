# Scrum
A simple scrum board using node.js

# Getting started

**1st method**
Clone the repo and run **node index.js** 
add hostname and port as optional arguments eg **node index.js 0.0.0.0 8000"**
**2nd method**
Clone the repo and run docker-compose up --build -d

# Requirements
node.js
